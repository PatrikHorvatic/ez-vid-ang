# Security Policy

## Supported Versions

Supported versions are the latest 2 major versions of the Angular framework.

| Version | Supported          |
| ------- | ------------------ |
| 21.2.x   | :white_check_mark: |
| 22.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you find a vulnerability, please do not post it on the forum.

You can contact me through email: patrik.horvatic.development@gmail.com
